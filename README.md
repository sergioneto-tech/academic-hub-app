# Academic Hub

O **Academic Hub** é uma aplicação pensada para **organizar a tua vida académica num único sítio** — cadeiras (UCs), anos/semestres, prazos, exames, apontamentos e progresso — com foco no que as ferramentas genéricas (Calendário/Excel/Sheets) normalmente não resolvem bem: **contexto, consistência e visão global**.

---

## Porque usar o Academic Hub (em vez de “remendos” com Calendar/Excel/Sheets)?

Ferramentas como **Google Calendar**, **Excel** ou **Google Sheets** são ótimas no que fazem… mas são **generalistas**:
- o **Calendar** é excelente para **marcar eventos**, mas não “entende” cadeiras, avaliações, dependências e progresso;
- o **Excel/Sheets** são excelentes para **tabelas e contas**, mas rapidamente viram “o ficheiro da verdade” que se multiplica em versões, colunas e abas (e perde-se o controlo).

O Academic Hub existe para juntar o melhor dos dois mundos: **planeamento + organização académica com estrutura própria**.

---

## Vantagens práticas do Academic Hub

### 1) Tudo com contexto (não só datas soltas)
Em vez de teres “Exame – 20h” no calendário e “Notas – UC X” numa folha, no Academic Hub a informação fica **ligada ao seu contexto académico**:
- cada tarefa/prazo pertence a uma UC/semestre;
- consegues ver rapidamente “o que tenho para esta cadeira?” e “o que falta neste semestre?”.

### 2) Visão global e prioridades reais
Spreadsheets são boas para listar, mas fracas para **priorizar**.  
O Academic Hub é orientado a:
- visão por **ano/semestre**;
- estado (por fazer / em curso / concluído);
- foco no **próximo passo**, sem andares a saltar entre apps.

### 3) Menos fricção, menos “gestão de gestão”
O “sistema” típico acaba assim:
- Calendar para datas  
- Sheets para listas  
- Notas no telemóvel  
- PDFs espalhados no Drive  
Resultado: **mais manutenção do sistema do que estudo**.

O Academic Hub reduz fricção ao centralizar o essencial (e manter o resto como anexos/links quando necessário).

### 4) Consistência e menos erros
Em Excel/Sheets é fácil:
- duplicar linhas,
- apagar fórmulas,
- trocar a versão do ficheiro,
- perder histórico.

Num hub académico, a estrutura é mais rígida e orientada ao caso de uso — o que **baixa o risco de “estragar a organização”**.

### 5) Acesso multi-dispositivo e sincronização
O Academic Hub foi pensado para correr como aplicação moderna (web), com autenticação e backend (ex.: Supabase), permitindo usar em vários dispositivos sem andares a enviar ficheiros de um lado para o outro.

## Comparação rápida

| O que precisas | Google Calendar | Excel / Sheets | Academic Hub |
|---|---|---|---|
| Marcar eventos e lembretes | ✅ forte | ⚠️ possível, mas fraco | ✅ (com contexto académico) |
| Estrutura por ano/semestre/cadeiras | ⚠️ manual | ⚠️ manual | ✅ nativo (pensado para isso) |
| Progresso e visão global | ⚠️ limitado | ✅ possível (mas dá trabalho) | ✅ foco em progresso e prioridades |
| “Uma verdade” (sem versões/ficheiros) | ✅ | ⚠️ fácil duplicar e perder controlo | ✅ centralizado |
| Personalizado ao estudo | ⚠️ não | ⚠️ depende de ti | ✅ por design |

> Nota honesta: **Calendar e Excel/Sheets continuam a ser excelentes** para casos específicos (agenda pura, análises complexas). O Academic Hub é para quando queres **um sistema académico coeso**, não um conjunto de ferramentas soltas.

---

## Para quem faz sentido?

- Estudantes que têm várias UCs por semestre e querem **clareza**.
- Quem já tentou organizar-se com folhas e calendários, mas sente:
  - “tenho tudo espalhado”  
  - “não sei o que é prioridade”  
  - “perco tempo a manter o sistema”.
